In this site we can learn some basics about HTML:

              HTML stands for Hyper Text Markup Language. HTML is the standard language for creating web pages and documents viewable in web browsers. It structures content using tags and attributes, defining elements like headings, paragraphs, links, and forms. HTML is the backbone of the World Wide Web.

             In 1989, Tim Berners-Lee, a British scientist, invented HTML while working at CERN . He designed it to facilitate information sharing and communication among researchers.
